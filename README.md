## Tạo Wordpress với Docker Compose


#### Clone source from GitHub

    $git clone https://github.com/Ducmy/docker-wordpress-dev
    $git checkout master

#### Install Docker & Docker-Sync
    
#### Khởi động docker-compose:
    $docker-sync start

#### Tắt docker-compose:

    $docker-compose down

#### URL wordpress:
http://localhost
 

#### URL phpadmin:
http://localhost:8080

#### Database User:
    user:     root
    password: root
